About Atom

Atom is an open source text editor, developed by the GitHub team

Atom was designed to be customizable to the core, while retaining a focus on user-friendliness.

Github Integration

Version control is an important aspect of any project and Atom comes with basic Git and GitHub integration baked in.

//keyboard shortcut: "alt-G O" - open on github



Snippets

Snippets are an incredibly powerful way to quickly generate commonly needed code syntax from a shortcut.

Packages

Packages can be incredibly powerful and can change everything from the very look and feel of the entire interface to the basic operation of core functionality.

Minimap

Minimap displays a condensed view of your code for quick navigation.

Git-Control

Provides a GUI interface to manage all commonly-used git commands.

Atom HTML Preview

A live preview tool for Atom

Pigments

A package to display colors in project and files
